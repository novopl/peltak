class peltak.core.versioning.VersionStorage(version_file: str)[source]¶

Base class for version storage.

A version storage is a way to store the project version. Different projects will have different ways of storing the version. The simplest case is a text file that holds just the version number (probably called VERSION). Python projects can also use a version stored as __version__ variable inside the project package/module. A Node.js project on the other hand will probably keep the version in package.json. All of the above strategies can be (and are) implemented through subclassing this class.

@see PyVersionStorage, RawVersionStorage, NodeVersionStorage
