    class peltak.core.git.BranchDetails(name, type=None, title=None)[source]¶

    Branch name parsed into type and title.

    Helpful for things like implementing git flow etc.

    name¶

    The full name of the branch as returned by git.

    Type

    str

    type¶

    The type of the branch. This assumes the type/title branch naming format and will be the text before / in the branch name.

    Type

    str

    title¶

    The title of the branch. This assumes the type/title branch naming format and will be the text after / in the branch name.

    Type

    str

    author¶

    The commit author.

    Type

    title¶

    The title of the commit. This is the first line of the commit message and should usually be shorter than 50 characters.

    Type

    str

    desc¶

    The commit description. This is the portion following the title. Will be an empty string if the commit contains only the title.

    str

    Return details about a given commit.

    Parameters

    sha1 (str) – The sha1 of the commit to query. If not given, it will return the details for the latest commit.

    Returns

    Commit details. You can use the instance of the class to query git tree further.

    Return type

    property number¶

    Return this commits number.

    This is the same as the total number of commits in history up until this commit.

    This value can be useful in some CI scenarios as it allows to track progress on any given branch (although there can be two commits with the same number existing on different branches).

    Returns

    The commit number/index.

    Return type

    int

    Parents of the this commit.

    peltak.core.git.guess_base_branch()[source]¶

    Try to guess the base branch for the current branch.

    Do not trust this guess. git makes it pretty much impossible to guess the base branch reliably so this function implements few heuristics that will work on most common use cases but anything a bit crazy will probably trip this function.

    Returns

    The name of the base branch for the current branch if guessable or None if can’t guess.

    Return type

    Optional[str]
